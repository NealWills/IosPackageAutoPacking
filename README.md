# IosPackageAutoPacking
IosPackageAutoPacking


### 1.放在项目根目录
### 2.cd到Autopacking目录
### 3.修改 autopacking.sh 内的参数
### 4.XCode打包一次，改生成的plist文件名，替换掉plist文件夹内的相关plist文件
### 5.终端执行命令 sh autopacking.sh
### 6.做出相应的选择,等待打包完成
### 7.获取ipa文件，万事大吉！！！！
